void JuliaFordContourTreeApp.TreeNode.RemoveFromTree ( )

Removes this node from whatever tree it appears in by attatching its parent to its child.

This function assumes that this node is non-critical, meaning it has exactly one parent and exactly one child. The function will not work properly if this is a critical node. However, the function does no error-checking, as the error-checking would be unreasonably expensive.
